O que é DOM?

  • - DOM - Document Object Model - é uma interface de programação que representa o conteúdo da página para que possa ser alterada.
  • - O DOM não é uma linguagem de programação, mas sem ele, o JavaScript não teria acesso ao HTML, ele é usado pelo JavaScript para acessar o documento e seus elementos.
  • - O conteúdo da página é armazenado no DOM e pode ser acessado e manipulado via JavaScript
  • - Todas as propriedades, métodos e eventos disponíveis para manipular e criar páginas da Web são organizados em objetos.
  • ~No início, o JavaScript e o DOM estavam fortemente interligados, mas, eventualmente, evoluíram para entidades separadas.~
  • ~O DOM foi projetado para ser independente de qualquer linguagem de programação específica~

O que são os objetos "window" e "document"?

  • - São os objetos mais utilizados
  • - De forma simples, o objeto window representa algo como o browser, e o objeto document é a raiz de todo o documento em si.
  • - O objeto window representa uma janela que contém um elemento DOM.
  • - O objeto document representa o documento HTML fonte da página.
  • - Browser Object Model (BOM) - permite que o JavaScript "converse" com o browser.

    • Exemplos:

  • - window.close() - fecha a janela atual
  • - window.open() - abre uma nova janela
  • - window.moveTo() - move janela atual
  • - window.resizeTo() - reajusta o tamanho da janela
  • - Document Object Model (DOM) - permite que o JavaScript modifique o código HTML.

    • Exemplos:

  • - document.getElementsByClassName(className) - Retorna um vetor de objetos com todos os elementos que possuem a classe
  • - Document.getElementsByTagName(tagName) - Retorna um vetor de objetos com todos os elementos com a tag

getElementById, getElementsByClassName, querySelector ou querySelectorAll?

  • - getElementById - Retorna a referência do elemento através do seu ID - O valor desse atributo deve ser único dentro do documento. (Elementos que não estão no documento não são procurados por getElementById. Quando criar um elemento e atribuir um ID ao mesmo, você deve inserir o elemento na árvore do documento)
  • - getElementsByClassName - Retorna um vetor de objetos com todos os elementos filhos que possuem o nome da classe dada. Funciona apenas com classes.
  • - querySelector - Retorna o primeiro elemento dentro do documento que corresponde ao grupo especificado de seletores. Funciona para qualquer tipo de seletor - id, class, atributo.
  • - querySelectorAll - Funciona como o querySelector, porem retorna uma nodeList com todos os elementos que possuem o seletor procurado (Funciona para quanquer tipo de seletor).

O que é um Element no JavaScript?

  • - Element é um tipo específico de nó (node.ELEMENT_NODE). É um nó escrito usando uma tag no documento HTML.

    • Outros tipos de nós:

  • - Attt - Representa um atributo
  • - Text - Representa conteudo textual em um elemento ou atributo
  • - Comment - Representa um comentário

innerHTML, innerText ou textContent?

  • - innerHTML - Retorna todo o conteúdo HTML escrito dentro do nó buscado exatamente como está escrito.
  • - innerText - Retorna apenas o texto escrito sem as tags e com espaçamento normal entre cada uma das palavras, como um texto simples.
  • - textContent - Retorna todo o texto escrito sem as tags mas mantendo o texto e espaçamento exatamente igual ao achado.

Qual a diferença entre uma NodeList e uma array comum? Como transformar a NodeList em uma array?

  • - Objetos NodeList são coleções de nodos semelhantes aos objetos retornados pelos métodos Node.childNodes e document.querySelectorAll() (Ou seja, coleções de objetos);
  • - O Nodelist é um objeto apenas semelhante a um Array, e por isso métodos específicos para arrays não funcionarão no NodeList.
  • - A melhor maneira de se transformar uma nodeListe em array é atraves do Array.from(nodeList).

Como criar elementos via JavaScript com document.createElement e element.appendChild

  • - Document.createElement() cria o elemento HTML especificado ou um HTMLUnknownElement se o nome do elemento dado não for conhecido.
  • - Sintaxe --- let elemento = document.createElement(nomeDaTag);
  • - element.appendChild() - Adiciona um nó ao final da lista de um pai especificado
  • - Sintaxe --- let filho = elemento.appendChild(filho);
